Integration Guide

How to Add Google Reviews to Your Webflow Site

Follow these steps to integrate Google Reviews into your Webflow CMS project. Works with Webflow CMS, Webflow Ecommerce, and static sites.

  1. Get a Google Places API Key: Go to Google Cloud Console, create a project, enable the Places API, and generate an API key.
  2. Find your Place ID: Use the Places API explorer to get your business Place ID.
  3. Deploy to Netlify: Push this repo to GitHub, connect to Netlify, and add environment variables:
    • GOOGLE_PLACES_API_KEY = your API key
    • PLACE_ID = default place ID (optional, fallback if data-placeid not set)
    • REVIEWS_LANGUAGE (optional, default en; overridden by data-locale)
  4. Embed in Webflow: In your Webflow page, add a custom code block with: 
    <div id="google-reviews-widget"
     data-endpoint="https://your-site.netlify.app/.netlify/functions/google-reviews"
     data-placeid="ChIJN1t_tDeuEmsRUsoyG83frY4"
     data-layout="carousel"
     data-mode="dark"
     data-max="8"
     data-min-rating="4"
     data-autoplay="6000"
     data-locale="en">
</div>
<link rel="stylesheet" href="https://your-site.netlify.app/assets/css/google-reviews-widget.css">
<script src="https://your-site.netlify.app/assets/js/google-reviews-widget.js" defer></script>
  5. Customize: Adjust these data attributes:
    • data-placeid: Google Place ID (optional, overrides env PLACE_ID)
    • data-layout: carousel or grid
    • data-mode: dark or light
    • data-max: number of reviews (default 10)
    • data-min-rating: min rating to show (default 0)
    • data-autoplay: ms delay for carousel (0 = disabled)
    • data-locale: BCP-47 language tag (e.g., es, en)
