# Agents & Roles

- Maintainer: triage issues, review PRs, keep dependencies minimal
- Frontend: evolve widget UX (carousel, grid, accessibility, SEO)
- Backend: maintain Netlify Function, rate limits, error handling
- Docs: keep README/usage up to date, examples verified

## Contribution Tips
- Prefer functional programming style in JS
- Keep bundle lightweight (no heavy deps); vanilla JS where possible
- Add tests for data transforms if logic grows
- Respect API quotas; cache when possible
